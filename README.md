# 職安助手 Agent — OSHA Auditor

台灣營造工地職業安全衛生智能稽核系統

## 功能特色

- **智能稽核分析** — 上傳工地現場照片或書面文件，AI 自動辨識並進行法規符合性分析
- **批次分析模式** — 一次上傳最多 10 個檔案進行批量分析
- **法規查詢系統** — 整合《職業安全衛生法》、《營造安全衛生設施標準》等重要法規
- **專業督導報告** — 自動產出包含缺失判定、法規條文引用的稽核報告

## 支援檢查類型

### 現場工地照片
- 開口防護（電梯井、樓梯口、預留孔洞）
- 施工架安全（交叉拉桿、腳趾板、壁連座）
- 高處作業（安全帶使用、掛點設置）
- 臨時用電（電箱、漏電斷路器、電線防護）

### 書面文件
- 危害告知紀錄、協議組織會議紀錄
- 職業安全衛生管理計畫、工作守則
- 自動檢查紀錄、教育訓練紀錄
- 人員證照、勞工名冊、保險資料

## 技術架構

- **前端框架**: React 19 + TypeScript
- **建構工具**: Vite 7
- **UI 元件**: shadcn/ui + Radix UI
- **樣式**: Tailwind CSS 3
- **圖表**: Recharts

## 快速開始

```bash
# 安裝依賴
npm install

# 啟動開發伺服器
npm run dev

# 建構生產版本
npm run build

# 預覽生產版本
npm run preview
```

## 專案結構

```
src/
├── components/          # 元件
│   ├── ui/              # shadcn/ui 基礎元件 (40+)
│   ├── FileUploader.tsx       # 單檔上傳
│   ├── BatchFileUploader.tsx  # 批次上傳
│   ├── AnalysisResult.tsx     # 分析結果展示
│   ├── BatchAnalysisResult.tsx # 批次分析結果
│   └── RegulationSearch.tsx   # 法規查詢
├── data/                # 法規資料庫
├── hooks/               # 自訂 Hooks
├── types/               # TypeScript 型別定義
├── App.tsx              # 主應用元件
├── main.tsx             # 入口點
└── index.css            # 全域樣式
```
